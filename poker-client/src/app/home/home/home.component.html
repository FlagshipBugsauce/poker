Welcome

This is an experimental online, multiplayer, poker application. This application was made using Spring Boot, Angular, MongoDB and will be deployed to AWS (S3 for client, elastic beanstalk for backend).

Currently, the poker game has not yet been implemented. In its place is a simple card drawing game. The complexity of this simple game will steadily increase until it becomes a proper NL Hold'em sit n go style game. Once this point is reached, it may evolve further into other card games, cash games, etc...
