How to Create a Game

To create a game, simply fill out the form below and click the create button. You will be taken to the game lobby where you can wait for more players to join.

  • You must provide a name, specify the number of players, and a buy-in, for the game you're creating.
  • You can start the game at any point when all players have clicked ready.
  • The game doesn't need to be full in order to start.
  • You need at least 2 people in the game before you can start.
